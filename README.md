# ePortfolio Website

## Overview

This repository contains a **personal ePortfolio website** developed to showcase skills, projects, achievements, and professional information.

It is designed to provide a digital profile for academic or professional presentation.

---

## Features

- Personal Profile Section
- Skills Showcase
- Project Gallery
- Resume / CV Display
- Contact Form
- Responsive Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap (optional)

---

## Sections Included

1. Home
2. About Me
3. Skills
4. Projects
5. Resume
6. Contact

---

## Installation

1. Download repository: git clone https://github.com/pyatrick666/ePortfolio.git 
2. Open: index.html 
3. View in browser.
